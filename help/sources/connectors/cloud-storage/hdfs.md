# (Beta) Connettore Apache HDFS

Adobe Experience Platform offre connettività nativa per fornitori di cloud come AWS, Google Cloud Platform e Azure, consentendo di portare i dati da questi sistemi. I dati contenuti possono essere formattati come JSON, parquet o delimitati. Il supporto per i fornitori di archiviazione cloud include Apache HDFS.

La documentazione seguente fornisce informazioni su come collegare Apache HDFS alla piattaforma utilizzando le API o l&#39;interfaccia utente:

## Connect Apache HDFS alla piattaforma mediante API

- [Esplora un sistema di archiviazione cloud utilizzando l&#39;API del servizio di flusso](../../tutorials/api/explore/cloud-storage.md)
- [Raccolta di dati di archiviazione cloud tramite l&#39;API del servizio di flusso](../../tutorials/api/collect/cloud-storage.md)
