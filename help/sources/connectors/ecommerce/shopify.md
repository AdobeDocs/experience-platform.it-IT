# [!DNL Shopify] connettore

Adobe Experience Platform consente di acquisire dati da origini esterne e allo stesso tempo di strutturare, etichettare e migliorare i dati in arrivo tramite [!DNL Platform] servizi. È possibile acquisire dati da diverse origini, ad esempio applicazioni Adobe, archiviazione basata su cloud, database e molte altre.

[!DNL Experience Platform] fornisce supporto per l’acquisizione di dati da un’origine E-Commerce. Il supporto per i provider di e-commerce include [!DNL Shopify].

La documentazione seguente fornisce informazioni sulle modalità di connessione [!DNL Shopify] a [!DNL Platform] utilizzando le API o l’interfaccia utente:

## Connetti [!DNL Shopify] a [!DNL Platform] utilizzo delle API

- [Esplorare le tabelle di dati utilizzando l’API del servizio Flusso](../../tutorials/api/explore/tabular.md)
- [Creare un flusso di dati per un’origine eCommerce utilizzando l’API del servizio di flusso](../../tutorials/api/collect/ecommerce.md)
