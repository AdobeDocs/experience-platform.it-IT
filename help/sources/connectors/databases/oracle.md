# Connettore Oracle

Adobe Experience Platform offre connettività nativa per provider di database come Microsoft, MySQL e Azure. È possibile trasferire i dati da questi sistemi in Piattaforma.

Sono supportati diversi tipi di database di terze parti, tra cui relazionale, NoSQL o data warehouse. Il supporto per i provider di database include Oracle.

La documentazione seguente fornisce informazioni su come collegare Oracle alla piattaforma utilizzando le API o l&#39;interfaccia utente:

- [Creare un connettore Oracle utilizzando l&#39;API di servizio di flusso](../../tutorials/api/create/databases/oracle.md)

- [Raccolta di dati da un database tramite l&#39;API del servizio di flusso](../../tutorials/api/collect/database-nosql.md)
