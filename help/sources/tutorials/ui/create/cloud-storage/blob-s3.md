# Creare un [!DNL Azure Blob] o [!DNL Amazon] Connessione sorgente S3 nell’interfaccia utente

I connettori di origini in Adobe Experience Platform consentono di acquisire dati di origine esterna in base a una pianificazione.

Le origini di archiviazione cloud possono inserire i tuoi dati in [!DNL Platform] senza dover scaricare, formattare o caricare. I dati acquisiti possono essere formattati come XDM JSON, XDM Parquet o delimitati. Ogni passaggio del processo viene integrato nel flusso di lavoro Origini. [!DNL Platform] consente di inserire dati da [!DNL Azure Blob] e [!DNL Amazon S3] attraverso i batch.

Per istruzioni dettagliate sull&#39;utilizzo di [!DNL Azure Blob] connettore di origine, leggere le [[!DNL Azure Blob] guida del connettore di origine](./blob.md).

Per istruzioni dettagliate sull&#39;utilizzo di [!DNL Amazon S3] connettore di origine, leggere le [[!DNL Amazon S3] guida del connettore di origine](./blob.md).
