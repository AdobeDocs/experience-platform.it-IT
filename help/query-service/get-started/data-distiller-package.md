# Pacchetto Data Distiller

Data Distiller è un pacchetto che include un sottoinsieme delle funzionalità di Adobe Experience Platform. Questo pacchetto è destinato alla preparazione dei dati di post-acquisizione per i casi di utilizzo in tempo reale di profili cliente o dati analitici tramite query batch eseguite in Query Service. Per ulteriori dettagli sul pacchetto Data Distiller, consulta la [Panoramica di Data Distiller](../data-distiller/overview.md).
