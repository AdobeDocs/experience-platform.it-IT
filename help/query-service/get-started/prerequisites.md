# Guida introduttiva a Adobe Experience Platform [!DNL Query Service] {#getting-started}

Utilizza Adobe Experience Platform Query Service per eseguire query SQL sui set di dati acquisiti, unire dati da più origini e generare set di dati derivati per analisi, flussi di lavoro di apprendimento automatico o Profilo cliente in tempo reale. Dopo aver acquisito i dati, accedi a Query Service tramite l’interfaccia utente per l’analisi interattiva e la collaborazione oppure tramite l’API per l’esecuzione automatizzata e programmatica delle query.

## Prerequisiti {#prerequisites}

Prima di iniziare a eseguire query sui dati, assicurati di disporre di:

- **Autorizzazioni richieste**: il tuo account utente ha accesso a Query Service in Experience Platform. Se il servizio non è disponibile nell&#39;interfaccia utente, consulta la [documentazione sulle autorizzazioni](../../access-control/home.md#permissions) e contatta l&#39;amministratore di sistema.
- **Acquisizione dei dati**: sono stati acquisiti dati in Experience Platform.

Se devi acquisire dei dati, guarda il video [tutorial sull&#39;acquisizione dei dati](https://experienceleague.adobe.com/docs/platform-learn/tutorials/data-ingestion/create-datasets-and-ingest-data.html) per una panoramica sulla creazione di set di dati, la mappatura dello schema, l&#39;acquisizione e la convalida. Leggi la [documentazione sulla panoramica dell&#39;acquisizione](../../ingestion/home.md) per informazioni più approfondite e collegamenti ad altre risorse di apprendimento.

## Percorsi di avvio rapido

Dopo aver acquisito i dati in Experience Platform, puoi iniziare a utilizzare Query Service utilizzando [!DNL Query Editor] in Experience Platform o l&#39;API Query Service.

### [!DNL Query Editor]

Utilizza [!DNL Query Editor] per l&#39;analisi, l&#39;esplorazione dei dati e lo sviluppo di query collaborative. Per una panoramica delle funzionalità dell&#39;interfaccia utente, consulta la [documentazione dell&#39;interfaccia utente di Query Service](../ui/overview.md). Per informazioni su come scrivere ed eseguire query nell&#39;interfaccia utente, leggere [[!DNL Query Editor user guide]](../ui/user-guide.md).

### API servizio query

Utilizza l’API Query Service per flussi di lavoro automatizzati, gestione di modelli di query e integrazioni programmatiche. Per istruzioni dettagliate sull&#39;utilizzo dell&#39;API di Query Service, consultare la [Guida per gli sviluppatori di Query Service](../api/getting-started.md).
