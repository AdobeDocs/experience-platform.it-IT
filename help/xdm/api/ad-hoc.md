# Schemi ad hoc

In circostanze specifiche, può essere necessario creare un [!DNL Experience Data Model] (XDM) con campi a cui viene assegnato un namespace per l’utilizzo solo da un singolo set di dati. Questo schema è denominato schema ad hoc. Gli schemi ad hoc vengono utilizzati in diversi flussi di lavoro di acquisizione dati, tra cui l’acquisizione di file CSV e la creazione di alcuni tipi di connettori di origine. Per ulteriori informazioni sugli schemi ad hoc, tra cui i passaggi per crearne uno nella sezione [!DNL Schema Registry] API, fare riferimento al [esercitazione sugli schemi ad hoc](../tutorials/ad-hoc.md).
