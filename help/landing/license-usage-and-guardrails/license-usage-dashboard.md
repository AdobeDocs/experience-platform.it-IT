# [!UICONTROL Utilizzo licenze] dashboard {#license-usage-dashboard}

L’interfaccia utente di Adobe Experience Platform fornisce una dashboard attraverso la quale puoi visualizzare informazioni importanti sull’utilizzo delle licenze della tua organizzazione, acquisite durante un’istantanea giornaliera.

Per istruzioni dettagliate su come accedere e interagire con la dashboard Utilizzo licenze nell’interfaccia utente, nonché per ulteriori informazioni sulle metriche disponibili visualizzate nella dashboard, visita [guida del dashboard utilizzo licenze](../../dashboards/guides/license-usage.md).

Per una panoramica di tutte le funzioni della dashboard in Experience Platform, leggi [panoramica delle dashboard](../../dashboards/home.md).
