# Monitorare i flussi di dati

[!UICONTROL Destinazioni] consente di attivare i dati da Adobe Experience Platform a innumerevoli partner esterni.

I flussi di dati sono una rappresentazione dei processi di dati che consentono di spostare i dati in [!DNL Platform]. Questi flussi di dati sono configurati tra servizi diversi, aiutando a spostare i dati dai connettori di origine ai set di dati di destinazione, dove vengono quindi utilizzati da [!DNL Identity Service] e[!DNL Real-Time Customer Profile] prima di essere infine attivato sulle destinazioni.

Per ulteriori informazioni sul monitoraggio dei flussi di dati per le destinazioni, consulta la sezione [Monitorare i flussi di dati per le destinazioni nell’interfaccia utente](../../dataflows/ui/monitor-destinations.md) esercitazione.
