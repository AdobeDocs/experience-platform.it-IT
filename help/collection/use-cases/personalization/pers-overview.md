# Panoramica sui casi di utilizzo di Personalization

Adobe Experience Platform Web SDK consente alle aziende di fornire un’ampia varietà di casi di utilizzo di personalizzazione alle proprie proprietà web. Puoi collegare facilmente i processi di raccolta dati, decisioning e attivazione tra le soluzioni Adobe come Target, Journey Optimizer e Audience Manager.

SDK supporta architetture flessibili, tra cui modelli lato client, lato server e ibridi, consentendo di recuperare ed eseguire il rendering di offerte personalizzate che soddisfano la maggior parte dei casi d’uso dell’organizzazione.

## Esempi di Web SDK

Oltre alle pagine dei documenti in questa cartella, Adobe gestisce un archivio di applicazioni di esempio a cui puoi fare riferimento. Consulta [Esempi di Web SDK](https://github.com/adobe/alloy-samples/) su GitHub per ulteriori casi di utilizzo di personalizzazione, tra cui:

* Personalizzazione lato client
* Personalizzazione lato server
* Personalizzazione ibrida
* Personalization con applicazioni a pagina singola
