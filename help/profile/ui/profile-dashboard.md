# (Beta) [!UICONTROL Profiles] dashboard {#profile-dashboard}

>[!IMPORTANT]
>
>La funzionalità del dashboard è attualmente in versione beta e non è disponibile per tutti gli utenti. La documentazione e le funzionalità sono soggette a modifiche.

L’interfaccia utente di Adobe Experience Platform fornisce una dashboard attraverso la quale è possibile visualizzare informazioni importanti sui dati [!DNL Real-time Customer Profile] acquisiti durante un’istantanea giornaliera.

Per istruzioni dettagliate su come accedere e interagire con il dashboard [!UICONTROL Profiles] nell&#39;interfaccia utente e per ulteriori informazioni sulle metriche disponibili visualizzate nel dashboard, visita la [[!UICONTROL Profiles] guida del dashboard](../../dashboards/guides/profiles.md).

Per una panoramica di tutte le funzioni del dashboard all&#39;interno dell&#39;interfaccia utente di Experience Platform, leggere la [panoramica delle dashboard](../../dashboards/home.md).
