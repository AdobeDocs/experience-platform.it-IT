# Note sulla versione di Adobe Experience Platform

**Data di rilascio: 12 febbraio 2020**

Aggiornamenti alle funzioni esistenti in Adobe Experience Platform:

* [Origini](#sources)

## Origini {#sources}

Adobe Experience Platform può acquisire dati da sorgenti esterne e allo stesso tempo strutturare, etichettare e migliorare tali dati utilizzando [!DNL Platform] servizi. È possibile acquisire dati da diverse sorgenti, come applicazioni di Adobe, archiviazione basata su cloud, software di terze parti e il sistema CRM in uso.

[!DNL Experience Platform] fornisce un’API RESTful e un’interfaccia utente interattiva che consente di impostare facilmente le connessioni sorgente per vari provider di dati. Queste connessioni di origine ti consentono di autenticare e connettersi a sistemi di archiviazione esterni e servizi CRM, impostare i tempi di esecuzione dell’acquisizione e gestire il throughput di inserimento dei dati.

**Nuove funzioni**

| Funzione | Descrizione |
| ------- | ----------- |
| Supporto API per SaaS - Sistemi di servizio | Nuovi connettori sorgente per [!DNL Salesforce Service Cloud] API e [!DNL ServiceNow] API. |
| Supporto API per SaaS - Sistemi di marketing | Nuovi connettori sorgente per [!DNL HubSpot] API. |
| Supporto API per [!DNL NoSQL] sistemi di database | Nuovi connettori sorgente per [!DNL AWS Redshift], [!DNL Google BigQuery], [!DNL MariaDB], [!DNL MySQL], [!DNL PostgreSQL]e [!DNL SQL Server] API. |
| Supporto API per sistemi di archiviazione cloud | Nuovi connettori sorgente per [!DNL Azure Data Lake Service Gen 2] API e [!DNL Google Cloud Storage] API. |
| Supporto dell’interfaccia utente per i sistemi di archiviazione cloud | Nuovo connettore sorgente per [!DNL Google Cloud Storage] nell’interfaccia utente di . |
