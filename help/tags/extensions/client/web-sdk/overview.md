# Panoramica dell’estensione tag Adobe Experience Platform Web SDK

L’estensione tag Adobe Experience Platform Web SDK invia i dati a Adobe Experience Cloud dalle proprietà web tramite Adobe Experience Platform Edge Network. L&#39;estensione Experience Platform Web SDK consente lo streaming dei dati nella piattaforma, la sincronizzazione delle identità, il consenso e la raccolta automatica dei dati contestuali.

Consulta la [Guida introduttiva](getting-started.md) per scoprire come impostare e distribuire questa estensione tag per iniziare a raccogliere dati.

Guarda il video seguente per vedere Adobe Experience Platform [!DNL Web SDK] e [!DNL Edge Network] in azione, utilizzando una singola chiamata per inviare dati a [!DNL Experience Platform], [!DNL Analytics], [!DNL Audience Manager] e [!DNL Target].
