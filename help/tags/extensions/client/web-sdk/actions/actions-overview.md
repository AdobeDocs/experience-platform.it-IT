# Tipi di azioni

In questa pagina sono elencate tutte le azioni supportate all’interno di una regola di tag. Per creare o modificare le azioni all’interno di una regola di tag:

1. Accedi a [experience.adobe.com](https://experience.adobe.com) utilizzando le credenziali Adobe ID.
1. Passa a **[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**.
1. Seleziona la proprietà tag desiderata.
1. Passa a **[!UICONTROL Rules]**, quindi seleziona la regola desiderata.
1. In [!UICONTROL Actions], selezionare un&#39;azione esistente o crearne una.
1. Imposta il campo a discesa [!UICONTROL Extension] su **[!UICONTROL Adobe Experience Platform Web SDK]**, quindi seleziona il [!UICONTROL Action Type] desiderato.

L’estensione tag Web SDK offre le azioni seguenti:

* [**[!UICONTROL Apply propositions]**](apply-propositions.md): esegui il rendering delle proposte nelle applicazioni a pagina singola senza incrementare le metriche.
* [**[!UICONTROL Apply response]**](apply-response.md): eseguire un&#39;azione in base a una risposta di Edge Network.
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md): attiva manualmente una valutazione del set di regole.
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md): esporta l&#39;API multimediale legacy in un oggetto finestra.
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md): consente la condivisione di un identificatore visitatore tra i domini di cui sei proprietario.
* [**[!UICONTROL Send event]**](send-event.md): invia i dati evento ad Edge Network.
* [**[!UICONTROL Send media event]**](send-media-event.md): invia i dati multimediali ad Edge Network.
* [**[!UICONTROL Set consent]**](set-consent.md): imposta il consenso desiderato per il visitatore.
* [**[!UICONTROL Update variable]**](update-variable.md): modifica un elemento dati [variable](../data-element-types.md#variable).
